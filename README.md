# Product Management API

API RESTful để quản lý sản phẩm sử dụng Flask và SQLite.

## Cài đặt

1. Tạo môi trường ảo:
   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux/Mac
   venv\Scripts\activate  # Windows

2. Cài đặt dependencies:
pip install -r requirements.txt

3. Chạy ứng dụng:
./run.sh
